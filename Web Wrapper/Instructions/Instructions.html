Changing the internal webpage

Now you can show a local HTML5 webpage if your internet is down.
The app comes with a HTML5 with bootstrap compliant page.

  1. Open nointernet.html in any web editor. It is located in the Internal folder of the assets director in the project.

Any files you add, such as images or other css need to be in the same folder as nointernet.html
